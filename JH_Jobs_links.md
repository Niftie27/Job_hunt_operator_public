# (Crypto) Job Hunt Source Library

This file is the cleaned source library for the Job Hunt operator. It is separate from the tracker so the main pipeline stays focused on actual applications and outreach.

## 1. Scope and hygiene

Kept the links in source-first format so they can be reused by an operator, scraper, or manual search workflow.

## 3. Recommended use order (SUGGESTIONS)

1. **Crypto-specific boards** for direct role discovery.
2. **Direct company career pages** from `OTHER URL` and `CRYPTO COMPANIES` for higher-signal openings.
3. **LinkedIn company pages** for broader monitoring and alert-based discovery.
4. **X/Twitter accounts** for early or less formal hiring signals.
5. **General job boards / freelance platforms** as fallback volume channels.
